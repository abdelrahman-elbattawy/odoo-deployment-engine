# 🚀 Odoo SaaS Deployment Engine

A full multi-tenant SaaS platform for deploying Odoo using Docker.

# 📦 Versions

## v1 - Nginx Based
Simple deployment using manual reverse proxy.

👉 `/v1-nginx`

## v2 - Traefik Based
Advanced SaaS engine with automation and SSL.

👉 `/v2-traefik`

# 🧠 Features

- Multi-tenant isolation
- Per-client PostgreSQL
- Automated deployment
- Environment-driven configs
- Scalable architecture

---

# 🚀 Goal

Build a production-ready SaaS engine for Odoo deployments.
